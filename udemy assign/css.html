WELCOME TO CSS

Introduction to Cascading Style Sheets:

The Cascading Style Sheet describes different style of appearance HTML elements in a separate document. The HTML is generally used for the describing the content of the web page. The web browser will take care for layout of the web page.

Cascading Style Sheets Properties:

background-colorcolor
font-family font-size
paddingmargin
text-decorationline-height

Types of css

  1. Inline Styles: Inline Style sheets are coded in the body of the web page as an attribute of the HTML tag.
  2. Embedded Styles: In Embedded styles the styles are written in the header part of the HTML web page.s
  3. External Styles: Linked Styles are coded in a separate text file.
